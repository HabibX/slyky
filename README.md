# Slyky

> Payment reception platform for developers, freelancers, and businesses — powered by Stellar.

Slyky is being built to make receiving global payments simpler, faster, and more accessible, especially for underserved markets.

⚠️ **Currently under active development for the Stellar Journey to Mastery – Blue Belt Challenge.**

---

## 🚀 MVP Scope (Phase 1)

The first version focuses on Stellar testnet payments using **XLM** and **USDC**.

### Features in Progress

- REST API to create payment requests
- Unique Stellar address + memo per payment
- Automatic payment detection via Horizon streaming
- Hosted checkout page (QR code + live status)
- Payment history dashboard
- User feedback collection & iteration

**Status:** 🛠️ In Development  
**Current Focus:** Backend payment request API + Stellar testnet payment listener

---

## 🛠️ Tech Stack

- **Backend:** Node.js, TypeScript, Express, Prisma, PostgreSQL
- **Blockchain:** Stellar SDK, Horizon (Testnet)
- **Frontend:** React, Tailwind CSS, Vite
- **Queue / Jobs:** Bull + Upstash Redis
- **Deployment:** Vercel / Render
